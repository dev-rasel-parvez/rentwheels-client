RentWheels is a modern and responsive car rental web application built with React, Tailwind CSS, and React Router. It provides users with a seamless experience for browsing, renting, and managing vehicles online.

---

## üåü Main Features

- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices.
- **Dynamic Car Listings:** Browse cars by category, brand, or rental price.
- **User Authentication:** Secure login and registration with Firebase Authentication.
- **Booking System:** Users can book cars and view their bookings in real-time.
- **Toast Notifications:** Beautiful popups for success, errors, and updates using React-Toastify.
- **Protected Routes:** Certain pages (like Dashboard or Bookings) are accessible only to logged-in users.
- **Modern UI Components:** Built with Tailwind CSS and React icons for a clean and elegant interface.
